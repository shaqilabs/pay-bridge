# EasyPaisa / EasyPay – Notes

- Client: `src/EasyPaisa/EasyPaisaClient.php`
- Usage guide: `src/EasyPaisa/Usage Guide EasyPaisa.md`
- Example: `examples/EasyPaisa.php`

- REST APIs:
  - `easypay-service/rest/v4/inquire-transaction`
  - `easypay-service/rest/v4/initiate-ma-transaction`

For the official documentation, refer to the EasyPay/EasyPaisa merchant integration guide provided with your store/merchant account.
